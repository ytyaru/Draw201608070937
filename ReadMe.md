# このソフトウェアについて #

Draw201608070937は私個人が学習目的で作成したソフトウェアです。

GDI, GDI+, DirectX, による描画コードを書いてみました。
ただし実行されるのはGDIのみです。
いずれ実行中に切替ができるようにする予定です。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express

## ライブラリ ##

* GDI+
* DirectX9
